# 품목별 로케이션

![000](./img/010.png)

**품목별 로케이션은 창고, 상품별 지정 로케이션을 등록하는 기능입니다.**

---

## **1. 버튼별 기능**

- **공통버튼**
    - **검색**  선택된 검색 조건으로 검색하여 품목별로케이션 정보를 조회합니다
    - **저장**  선택된 로케이션에 대한 현재상태를 저장합니다
- **마스터 그리드 버튼**
    - **로케이션**  검색 입력창 좌측의 조건(보관로케이션 / 피킹로케이션 / 피킹완료로케이션)에 따른 로케이션을 조회합니다
    - **일괄적용**  로케이션 검색을 통해 조회된 로케이션을 선택된 창고별 품목 목록에 적용합니다
    - **업로드**  양식에 맞춰 업로드 시 창고별 품목을 신규 생성하거나 수정할 수 있습니다

---

## 2. **사전 작업**
- **마스터 그리드**

**[기준정보 > 기준정보 > 품목]** 에서 품목을 추가해야 합니다

---

## 3. **심화 기능**
- 등록된 로케이션(보관로케이션, 피킹로케이션, 피킹완료로케이션)이 있다면 등록여부가 등록으로 변경됩니다. 로케이션이 등록된 품목은 프로세스 진행 시 해당 로케이션이 적용됩니다.
